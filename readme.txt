SyngulARS is a revolutionary WordPress Theme builder.

== Description ==

Build WordPress themes without writing a single line of code.

== Installation ==

1. Upload `syngulars` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
